This project is a **full-featured electronics catalog website** built with **React**.
The application allows users to browse different categories of electronics, view detailed product information, and navigate between multiple pages in a smooth and intuitive way.
The project demonstrates **component-based architecture, client-side routing**, and structured state management in a real-world React application.

**Built with:**
- HTML5 for semantic structure
- SCSS for modular, maintainable styles
- BEM methodology for clean, structured code
- Responsive design that adapts beautifully to all screen sizes
- React
- TypeScript
- React Router

**Key Features:**
- Multi-page application with client-side routing
- Product catalog with categorized electronics
- Product details pages
- Responsive navigation menu
- Reusable React components
- Clean and scalable project structure
- Fully responsive layout for mobile, tablet, and desktop



**Getting Started:**
- Clone the repository:
  - git clone https://github.com/GumenchukAN/phone-catalog-react.git_
- Navigate to the project folder:
  - cd landing_bike-shop
- Install dependencies:
  - npm install_
- Run the project:
  - npm start_
